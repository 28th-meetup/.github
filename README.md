# 🍱 집밥도착 🍱

>  `집밥도착`은 해외 교민들이 저렴하고 간편하게 한식 집밥을 맛볼 수 있도록 반찬 및 반조리 식품을 판매하는 서비스입니다. 
>  한식 집밥을 판매하는 반찬가게와 교민들을 연결하여 한식을 쉽게 주문할 수 있도록 합니다.
